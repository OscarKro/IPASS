Detailed Description

Weatherstation Display class.

This class has functions needed to use the display from a KromWorks WeatherStation. It includes the Hardware library from Wouter van Ooijen called "hwlib.hpp". All the hardware communication is done in that library and all rights for hwlib are reserved for Wouter van Ooijen.
